XOR Gate

The XOR Gate performs a Logical Exclusive OR of the input signals.

Behavior (as an XOR):

IF an even number of inputs is 1 THEN the output is 0
IF an odd number of inputs is 1 THEN the output is 1

Behavior (as an XNOR):

IF an even number of inputs is 1 THEN the output is 1
IF an odd number of inputs is 1 THEN the output is 0

Invert output (Default False): True (XNOR) or False (XOR).
Number of inputs (Default 2): 2, 3, or 4.
