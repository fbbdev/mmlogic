Counter Device

The Counter Device is a simple 4 or 8 bit counter. You can gang counters together to build larger counters.

Behavior:

When the clock input changes from 0 to 1 the counter increments by 1. If the counter has reached its limit (15 decimal) it pulses in the Carry Out "C" signal. The Carry out signal can be used as the Clock to trigger another counter. So you can gang together as many counters as you like to build larger counters. The 4 outputs represent the current value of the counter. You can reset the counter at any time with the "R" set to 1. The Down "D" signal is used to set the direction of counting. If D is 0 the counter will count up, If D is 1 the counter will count Down.

Size (Default 4-bit): 4-Bit or 8-Bit
