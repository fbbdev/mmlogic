Analyze Tool

The analyze tool will allow you to examine the circuit for debugging purposes. It is currently NOT implemented completely and has no function yet.

Rules:

  • The analyze tool is only used while the simulator is running.
  • Depending on the eventual implementation, the analyze tool may be used to log to a file the time-stamped status of a node.

The analyze may use the same 6 symbols as the probe tool:

0 Analyze sees a 0 state.
1 Analyze sees a 1 state.
U Analyze sees a Unknown state.
- Analyze sees a Pull Down state.
+ Analyze sees a Pull Up state.
? Analyze sees a Float state.
