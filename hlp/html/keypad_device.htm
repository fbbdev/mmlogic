Keypad Device

The keypad device is a convenient input device that can generate one of 16 values.

Behavior:

When the user clicks on a key the corresponding key value will be presented to the Outputs.
IF a key is down (depressed with Left Mouse Button) E will Output 1.
otherwise E will Output 0.

The "E" Output should be used as an indicator to know when to use (consume) the output of the device.
