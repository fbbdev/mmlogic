Select Device

The select device is a 16 position rotary switch input device. It can be used as both an interactive input device or as an alternative to using Ground and Plus devices to hardwire some inputs by setting its initial state. For example is you wanted to lock the ALU device in compare mode set the Initial state to 4 and connect the Selector device to the ALU device function select lines.

Behavior:

When the user clicks on the select switch while the simulator is running it will:

IF Output n < F (hex) THEN Output a n+1.
IF Output n = F (hex) THEN Output a 0.

It will return to its initial state when the simulation is reset.
