ASCII Display Device

The ASCII Display device is an Output device that emulates a simple terminal device. The display will hold 16 characters across of 8 rows down.

Behavior:

IF R is 1 the screen is reset.
Otherwise the device will perform the following:

When the clock input "C" changes from 0 to 1 the device will perform the following:

IF the inputs evaluate to <CR> (Hex 0D) the cursor is moved to the next line.
IF the inputs evaluate to <BkSp> (Hex 08) it will erase the previous character.
Otherwise the character is appended to the current line.
IF you send more than 16 characters to a line it will return to column 1 and erase the line.
